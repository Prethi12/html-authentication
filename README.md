# HTML Authentication POC

This project is a simple Proof of Concept (POC) for an authentication system built using plain HTML.

## Pages Included

- login.html
- register.html
- forgot-password.html
- reset-password.html
- dashboard.html

## Features

- Login page with navigation to Register and Forgot Password
- Registration page with redirection to Login
- Forgot Password page with redirection to Reset Password
- Reset Password page with redirection to Login
- Dashboard page with Logout option

## Technologies Used

- HTML only (No CSS, No JavaScript)

## How to Run

1. Download or clone the repository
2. Open `login.html` in any web browser
3. Navigate using the provided links
